# Async Micro Project: Task Queue Manager

## Overview

A simple task queue manager that demonstrates practical asynchronicity patterns. This project simulates processing multiple tasks with different priorities and execution times.

## Features

- Task queue with priority support
- Parallel and sequential task execution
- Progress tracking
- Error handling
- Promise-based architecture

## How to Run

```bash
node index.js
```

## Project Structure

- `index.js` - Main application file
- `taskQueue.js` - Task queue implementation
- `README.md` - This file

## Learning Objectives

- Understanding Promise chains
- Managing concurrent operations
- Error handling in async code
- Building practical async utilities
